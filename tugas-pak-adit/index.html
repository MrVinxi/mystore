Analisis Topologi: Hubungan antar Router dan PC Client

Routing adalah proses kunci dalam jaringan komputer untuk menentukan jalur terbaik bagi paket data dari sumber ke tujuan. Berdasarkan topologi di atas, pemilihan metode routing sangat krusial.

Routing Statis

Jalur dikonfigurasi secara manual oleh administrator. Sangat stabil dan hemat resource hardware karena router tidak perlu memproses algoritma rute secara terus-menerus.

Routing Dinamis

Menggunakan protokol (seperti OSPF/EIGRP) untuk mencari jalur secara otomatis. Mampu melakukan pemulihan jalur (rerouting) secara instan jika ada kabel yang putus.
